# Tematica științifică

Modulul de **specialitate** acoperă cele **10 teme** din programa de examen. Fiecare temă este tratată
pe **trei planuri**, conform cerinței examenului:

1. **Teorie** — concepte, definiții, exemple de cod **C++**, complexitate, greșeli frecvente.
2. **Competențe vizate** — ce competențe formezi la elevi prin temă (raportat la programele școlare).
3. **Abordare metodică** — cum predai concret tema la clasă (metode, lecție, evaluare).

## Temele

::: tip Sfat de pregătire
Parcurge temele în ordine — primele patru (algoritmi, C++, metode de programare, structuri dinamice)
formează nucleul de programare pe care se sprijină grafurile și subiectele de tip „rezolvare de
probleme" de la proba scrisă.
:::

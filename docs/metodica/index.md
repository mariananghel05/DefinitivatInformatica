# Metodica predării disciplinelor informatice

Modulul de **metodică** acoperă cele **3 teme** din programă. Este zona în care examenul verifică nu
*ce* știi, ci *cum* predai și *cum* evaluezi. Conținutul este aliniat la **cadrul în vigoare**
(Legea 198/2023, curriculum centrat pe competențe, programe școlare actuale).

## Temele

| # | Temă | Conținut esențial |
|---|---|---|
| 1 | [Proiectarea didactică](/metodica/01-proiectare-didactica) | Curriculum, planificare, unitate de învățare, proiect de lecție |
| 2 | [Strategii didactice](/metodica/02-strategii-didactice) | Metode, tipuri de lecție, mijloace, forme de organizare |
| 3 | [Evaluarea randamentului școlar](/metodica/03-evaluare-randament) | Funcții, metode, itemi, notare |

::: tip Cum se leagă de specialitate
Fiecare temă din [tematica științifică](/stiintific/) are deja o secțiune de *abordare metodică*.
Acest modul oferă **cadrul teoretic** al metodicii, pe care îl aplici apoi la orice conținut de
informatică din subiectele de metodică ale examenului.
:::

::: warning Terminologie actuală
La proba scrisă se punctează folosirea **terminologiei actuale**: *competențe* (nu „obiective
operaționale" în sens vechi), *activități de învățare*, *evaluare formativă/sumativă*, *itemi* etc.
Programa din 2020 citează Legea 1/2011 — în conținut folosim cadrul **Legii 198/2023**.
:::
